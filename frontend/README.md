# AI 数据实践宝库 (AI Data Practice Treasure House)

聚焦 AI 领域实践能力搭建，整合框架、Agent、技能、知识库、应用等全链路资源，打造一站式学习与实操平台。

## 核心功能

- **AI 实践框架**: 覆盖主流 AI 框架，提供选型指南与实战案例。
- **Agent 相关**: 深入探索自主 Agent、Agent 平台及其部署教程。
- **Skill 相关**: 包含编程、AI 算法、前端/后端开发等技能学习路径。
- **知识库相关**: 整合论文、文档、视频教程等权威资源。
- **APP 应用相关**: AI 创作工具与行业应用测评。
- **网页相关**: 精选 AI 工具导航与社区资源。

## 技术架构

- **前端**: React 19 + TypeScript + Vite + Lucide React + React Router
- **设计风格**: 参照 Google Gemini 官网（深色/浅色主题、极简视觉层级）
- **路由**: 单页应用 (SPA) 架构，流畅切换无刷新

## 快速开始

```bash
cd frontend
npm install
npm run dev
```

## 设计规范

- **配色**: Gemini 深色/浅色主题
- **圆角**: 8px 统一圆角
- **动画**: 渐入显示、hover 缩放效果
